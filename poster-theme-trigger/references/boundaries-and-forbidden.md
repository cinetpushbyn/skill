# 边界与禁忌

## 本 Skill 的定位边界

`poster-theme-trigger` 是一个入口判断 Skill，不是完整策划器、生成器或审核器。

它只做：
- 识别触发来源（date / image_reference / inspiration）
- 返回基础主题起点信息

它不做以下任何事情：

---

## 禁止行为

### 1. 不生成完整海报方案

不写完整标题、副标题、卖点文案、正文、CTA。这些属于后续策划 Skill 的职责。

### 2. 不生成完整生图 prompt

不输出给 Midjourney、Stable Diffusion、DALL·E 等工具用的完整 prompt 描述。图片参考触发只做参考摘要。

### 3. 不做图片审核

不评价用户上传图片的好坏、优劣、是否适合、有没有问题。图片审核不是本 Skill 的触发场景。

### 4. 不做复杂主题优先级评分

不给主题方向打分、排序、做权重分析。只输出起点，不做筛选决策。

### 5. 不在普通日臆造节日

今天是普通日就返回普通日，不要强行拼凑出"某品牌日""随机美学节"之类的虚假节日。

### 6. 不擅自增加第四、第五类触发逻辑

本 Skill 只有三类触发：date / image_reference / inspiration。不要因为用户说了某个词就发明新类型。

### 7. 不把图片参考误判成直接生成指令

用户说"参考这张图做一张海报"不等于让你直接生图。这是参考触发，输出的是参考摘要，不是成品。

### 8. 不冒充后续策划 Skill

输出完基础起点信息后，明确告知用户这是起点，下一步需要进入策划 Skill 或生成 Skill 继续处理。

---

## 禁忌词

以下表达不允许出现在本 Skill 的任何输出中：

| 禁忌词 | 原因 |
|--------|------|
| 高级感拉满 | 空泛，无结构价值 |
| 氛围感十足 | 空泛，无结构价值 |
| 很有感觉 | 无法被下游解析 |
| 非常适合 | 没有具体指向 |
| 随便来几个 | 不严肃，降低输出质量 |
| 万能主题 | 不存在万能主题 |
| 直接出图 | 超出本 Skill 职责 |
| 一键生成完整海报 | 超出本 Skill 职责 |

---

## 越界风险提示

以下情况容易导致本 Skill 越界，需要特别注意：

- 用户说"顺便帮我写个文案" → 不做，告知需要切换到策划 Skill
- 用户说"直接出图就好了" → 不做，告知需要切换到生图 Skill
- 用户上传图片但只是让你评价 → 不触发，直接回应评价需求即可
- 用户要求做完整工作流 → 不做，这是系统设计需求，不是本 Skill 的触发场景
