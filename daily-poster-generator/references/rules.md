# rules.md — 视觉与场景约束规则

> 本文件定义所有硬约束与品牌规范。标注「硬约束」的条目不允许在提示词生成阶段被绕过或软化。

---

## 一、海报基础规格

| 参数 | 值 |
|------|----|
| 比例 | 9:16 竖版 |
| 格式 | PNG |
| 分辨率（建议） | 1080×1920px |

---

## 二、品牌露出规则

### 2.1 Logo 位置
- **硬约束**：Logo 固定在画面**左上角**体系内，不得移至其他位置
- 品牌形式：`RACHING 美晶`（中英文同时显示）
- 允许根据背景明暗自动微调 Logo 的大小与透明度
- 不允许将 Logo 做成颜色反转、镂空、分离中英文等变体处理

### 2.2 文字语言
- 内容文字（主标题、副标题、功能点）以**中文**为主
- 品牌名保留中英文形式，不做纯英文或纯中文处理

---

## 三、标题与文案规则

### 3.1 节日 / 节气海报标题

| 项目 | 内容 |
|------|------|
| 主标题 | 节点名称（如：谷雨、中秋） |
| 副标题 | 产品价值表达，**不写解释性长文案** |
| 主标题字数 | 2-4 字（节点名本身即可） |
| 副标题字数 | 6-10 字 |

副标题示例（合法）：
- `一柜锁住春茶香`（8 字）
- `为好茶留住时光`（7 字）
- `天然守护，茶香长存`（9 字）

副标题示例（不合法）：
- `RACHING 美晶智能茶叶柜，为您的春茶提供全方位保护`（过长、解释性）

### 3.2 日常品牌海报标题

- 使用品牌感短句，**不用节点名**，**不写解释性文案**
- 主标题：2-4 字，品牌感词组
- 副标题：6-10 字，品牌理念或价值主张

示例（合法）：
- 主标题：`时光醒茶` / 副标题：`守护每一克珍藏`
- 主标题：`静藏` / 副标题：`让好茶在最好的状态等你`

## 四、功能展示规则

### 4.1 四项功能点

| 功能点 | 核心关联场景 |
|--------|-------------|
| 智能控温湿 | 所有节气、梅雨季、回南天 |
| 隔潮防霉 | 梅雨季、回南天、春茶季 |
| 隔光抗氧 | 日照强的季节、夏季、日常 |
| 天然等离子 | 日常、高端礼赠场景 |

### 4.2 展示数量规则
- **硬约束**：节日 / 节气海报 → 展示最相关的 **1-2 个**功能点
- **硬约束**：日常品牌海报 → 展示**全部四项**功能点

### 4.3 功能文案风格
- **硬约束**：极简词组型，不写长解释句
- 合法示例：`智能控温湿` / `隔光抗氧`
- 不合法示例：`通过智能温控系统实时调节柜内温湿度，保护茶叶品质`

### 4.4 功能区视觉位置规则
- 功能展示区可自动判断位置
- **硬约束**：不得遮挡产品主体
- **硬约束**：不得破坏高级感
- **硬约束**：层级必须低于主标题（视觉权重：产品 > 主标题 > 副标题 > 功能点）

## 五、场景约束（硬约束）

以下规则全部为**硬约束**，不允许通过「场景创意自由」绕过：

### 5.1 产品环境底线
1. **禁止**将产品置于：泥地、积水、脏乱场所、露天低端环境
2. 使用室外场景时，**必须**为产品构建高品质承载介质或过渡空间，例如：
   - 防腐木地台
   - 高级石材地面（大理石、青石板、花岗岩）
   - 青石板观景台
   - 半开放茶亭（带顶、有遮蔽）
3. 极端天气场景（暴雨、大风、强烈日晒）**必须**优先室内化表达，或通过高端过渡空间处理

### 5.2 产品物理接地感（硬约束）
1. 产品**必须**具备接触阴影（contact shadow）
2. 产品**必须**体现合理的环境反射（环境光、地面材质反射）
3. 产品**必须**具备合理的柜体落位逻辑（不漂浮、不悬空）

### 5.3 产品主体地位（硬约束）
- **产品必须始终是画面主体**
- 任何辅助元素（节气氛围、背景、功能图标）的视觉权重均不得超越产品

### 5.4 允许的场景类型（第一版）
- 室内场景：书房、茶室、展厅、落地窗空间
- 半开放场景：茶亭、廊道、带顶阳台、室内外过渡空间
- 室外场景：须满足 5.1 约束，提供高品质承载介质

## 六、辅助元素规则

### 6.1 节日 / 节气辅助元素强度
- 允许：有明显主题氛围（如谷雨的雨纹/嫩芽、中秋的月影/灯笼光晕）
- 要求：适度丰富，不堆砌，不抢产品主体
- **禁止**：做成廉价节庆堆料图

### 6.2 全局黑名单（硬约束）

以下元素**绝对禁止**出现在任何模式的海报中：

| 禁止元素 | 禁止原因 |
|----------|----------|
| 人物 | 喧宾夺主，品牌调性不符 |
| 茶具堆砌 | 视觉混乱，与产品定位冲突 |
| 茶汤 | 品类误导，非茶饮品牌 |
| 假山 | 廉价园林感，调性低 |
| 杂乱植物 | 视觉噪声，破坏高级感 |
| 低端节庆装饰（廉价灯笼、彩纸、卡通气球） | 调性低 |
| 漂浮无接地感的产品 | 违反物理真实感 |
| 低质量拼贴感（明显抠图边缘、光影不一致） | 视觉质量不达标 |
| 廉价 PPT 排版感 | 专业度不足 |
| 过度卡通化图标 | 与品牌调性不符 |
| 实心色块廉价功能图标 | 降低画面品质感 |

---

## 七、版式层级规则

产品主体 > Logo > 主标题 > 副标题 > 功能展示区 > 辅助氛围元素

- 各层级视觉权重不允许倒置
- 主标题与副标题的位置可自动判断（上方/下方/居中），但不得遮挡产品主体
- 功能展示区的位置可自动判断，但不得与主标题同权重竞争
