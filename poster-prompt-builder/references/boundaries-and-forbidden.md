# boundaries-and-forbidden.md

这里说明这个 Skill 不该做什么，以及需要避免的行为和用词。

---

## 这个 Skill 不负责的事

| 任务 | 应该交给谁 |
|------|-----------|
| 给主题灵感 | poster-theme-trigger |
| 展开完整海报策划 | poster-brief-builder |
| 评估最终出图效果 | 不在工作流内 |
| 重新做一套完整 brief | poster-brief-builder |
| 输出多套风格完全冲突的方案 | 不做 |

---

## 禁忌行为

### 不要重新发明主题

如果用户已经有主题，不要说"我帮你重新想一个更好的方向"，直接用用户给的方向做 prompt。

### 不要把 prompt 写成海报方案说明书

prompt 不是策划文档，不要写"这张海报旨在表达……""设计理念是……"。

### 不要写成诗歌、散文、品牌宣言

prompt 要直接描述画面，不要文学化。

### 不要一边写 prompt 一边重新做策划

如果用户给的 brief 有不完整的地方，只做最小补全，不要展开成新的策划方向。

### 不要无节制堆风格词

4-6 个风格词足够，堆 10 个以上风格词会让模型混乱。

### 不要为了显得专业而塞大量摄影术语

比如"f/1.4光圈""ISO 100""85mm镜头"这类词，对商业海报生图帮助有限，不要随意堆砌。

### 不要照抄参考图

如果用户给了参考图，要融合风格，不能复制描述。

### 不要忽略产品主体

产品是主角，背景和场景是配角。不要让场景描述占了 80% 的 prompt 而产品只提一两个字。

### 不要把卖点写成长段广告词

卖点在 prompt 里只体现为排版意图，比如"底部信息区"，不要把"恒温储茶、保鲜锁味、智能感知"写成一段广告词塞进去。

### 不要输出无法直接给模型使用的废话

输出结果要能直接复制粘贴给生图模型，不需要用户二次整理。

---

## 禁忌词倾向

输出里尽量避免这类空泛表达：

- 高级感拉满
- 直接封神
- 绝美海报
- 质感炸裂
- 随便都能出好图
- 怎么都高级
- 越多元素越好
- 多堆一点氛围
- 参考图一模一样
- 完全复制这个风格

这类词没有画面信息，对生图模型没有帮助，写了反而干扰输出。

---

## 越界风险提示

如果遇到以下情况，不要直接处理，应该说明这不是本 Skill 的范围：

- 用户说"我还没想好主题，你帮我想"→ 转交 poster-theme-trigger
- 用户说"帮我重新策划一下这张海报"→ 转交 poster-brief-builder
- 用户说"你觉得出图效果好不好"→ 说明本 Skill 不负责审核
