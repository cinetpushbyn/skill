# output-format.md

这个 Skill 的输出格式固定为 JSON，包含两个字段。

---

## 标准格式

```json
{
  "image_prompt": "...",
  "negative_prompt": "..."
}
```

---

## 字段说明

### image_prompt

- 类型：字符串
- 语言：中文
- 长度建议：60-120 字，不超过 150 字
- 内容要求：包含产品主体、画面场景、视觉风格、色调、光线、构图、商业海报感
- 写法：写成一段自然流动的中文描述，不分行，不加标签

### negative_prompt

- 类型：字符串
- 语言：中文
- 长度建议：20-50 字
- 内容要求：覆盖常见风险项，用顿号分隔
- 写法：词语列表，不写成句子

## 不合格的输出示例

**不要这样写：**

```json
{
  "image_prompt": "一张高级感拉满的海报，质感炸裂，随便都能出好图，主题是谷雨，有茶叶柜，感觉很东方很高级",
  "negative_prompt": "不好的东西"
}
```

问题：image_prompt 充满空泛表达，没有具体画面信息；negative_prompt 无效。
