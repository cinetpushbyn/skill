# negative-prompt-rules.md

negative prompt 的作用是告诉模型"不要出现什么"，防止生图结果跑偏。

---

## 固定复用词（每次都加）

以下这些词适合固定出现在大多数海报的 negative prompt 里：

```
杂乱背景，廉价电商风，过度装饰，文字乱码，产品变形，模糊，噪点，过饱和颜色，强烈塑料感，低清晰度
```

---

## 按场景选用的词

### 产品类海报通用

```
产品比例错误，产品结构不清晰，背景喧宾夺主，产品主体模糊
```

### 有文字排版的海报

```
文字乱码，文字变形，字体花哨，文字与背景融合无法识别
```

### 东方美学 / 高端简约风格

```
廉价感，拼凑感，花哨装饰，多余元素，色彩脏乱，低端市集风
```

### 节气主题海报

```
过于卡通，儿童插画感，颜色过于鲜艳刺眼，元素堆砌
```

### 极简留白风格

```
画面拥挤，元素过多，背景复杂，色彩对比过强
```

---

## 不要写进 negative prompt 的内容

- 不要把正面诉求写成否定形式（比如不能写"不要黑色"，应该直接在 image_prompt 里写想要的颜色）
- 不要写太长，一般 20-40 个词足够
- 不要重复已经在 image_prompt 里强调过的正面内容

---

## 示例

标准海报 negative prompt：

```
杂乱背景，廉价电商风，过度装饰，花哨字体，文字乱码，产品变形，柜体比例错误，模糊，噪点，过饱和颜色，强烈塑料感，低清晰度，背景喧宾夺主
```
