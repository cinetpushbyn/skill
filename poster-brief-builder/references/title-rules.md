# title-rules.md — 标题写法规则

这里写主标题（`title`）和副标题（`subtitle`）的具体规则。

---

## 主标题规则

### 长度
- 推荐：8~14 字
- 最短不要低于 6 字，太短显得单薄
- 最长不超过 16 字，超过就放不进海报

### 风格
- 有意境，但不飘
- 有产品感，但不硬卖
- 可以用节气意象、自然意象，但要能跟产品搭上
- 中文古典短句风格适合东方茶仓类产品，但不要写成古文

### 好的标题长什么样

```
雨润百谷，茶藏有度          ✅ 有节气感，有产品功能暗示
春藏一柜，静养好茶          ✅ 简洁，产品感清楚
以时藏茶，以度护香          ✅ 意境感强，功能感也在
```

### 坏的标题长什么样

```
谷雨特惠，买一送一！        ❌ 促销横幅，掉价
全网最强茶叶储存神器        ❌ 太口号化，没质感
春天来了，茶香飘四方        ❌ 太泛，跟产品没关系
专业恒温恒湿，锁住每一分茶香  ❌ 太长，太像产品说明书
```

## 副标题规则

### 是否必填
可选。如果主标题已经很完整，不需要强行加副标题。

### 长度
- 推荐：16~24 字
- 不超过 28 字

### 作用
副标题补充主标题没说清楚的东西，通常是：
- 功能感的延伸
- 场景感的延伸
- 用户价值的延伸

### 好的副标题

```
让好茶在恰好的温湿之间静静陈放      ✅ 有场景感，有功能暗示，不促销
以东方留白呈现茶仓与春日空间的平衡感  ✅ 有美感，有品牌感
```

### 坏的副标题

```
立即购买，享受春日好茶体验！        ❌ 促销语，不该出现在 brief 里
我们用最专业的技术保护您的茶叶      ❌ 太硬，像客服话术
茶，值得被好好对待                  ❌ 太空，没有落地感
```

---

## 总结：标题的三条底线

1. 不促销（不出现「特惠」「立即购买」「买一送一」等词）
2. 不口号化（不写成像横幅标语的那种）
3. 不脱产品（意境再美，也要能跟产品搭上关系）
