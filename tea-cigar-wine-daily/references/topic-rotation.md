# Topic Rotation

Use this file to keep the daily encyclopedia varied.

## Knowledge Axes

Rotate axes within each category. Avoid using the same axis for the same category on consecutive daily outputs.

- definition: explain a core term
- craft: how something is made or processed
- region: place, terroir, origin, appellation, or production geography
- classification: grades, styles, legal categories, or labeling systems
- service: how it is served, opened, brewed, cut, lit, decanted, or poured
- storage: preservation conditions and why they matter
- etiquette: social use, hospitality, or responsible appreciation
- tasting-vocabulary: how to describe aroma, texture, finish, body, aftertaste
- history: verified historical development, route, institution, or named text
- misconception: correct a common false belief

## Snowball Rule

A topic can generate future related cards, but not immediately with the same angle.

Good sequence:

- Day 1: `cigar-wrapper-connecticut` / definition
- Day 2: `cigar-ring-gauge` / classification
- Day 3: `cigar-cutting-straight-v` / service

Weak sequence:

- Day 1: `cigar-spanish-cedar-box` / storage
- Day 2: `cigar-humidity-65-70` / storage
- Day 3: `cigar-humidor-airflow` / storage
